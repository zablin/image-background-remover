# 图片背景移除网站 MVP 需求文档


## 一、项目概述
项目名称：Image Background Remover
核心功能：用户上传图片，一键移除背景，下载透明背景图
技术栈：Next.js + Remove.bg API + Cloudflare Pages
目标用户：需要快速处理图片背景的设计师、电商从业者、内容创作者


## 二、MVP 核心功能


### 2.1 图片上传


#### 支持拖拽上传或点击选择


#### 支持格式：JPG、PNG、WEBP


#### 单次上传限制：最大 10MB


#### 上传后实时预览原图


### 2.2 背景移除


#### 点击"移除背景"按钮触发处理


#### 调用 Remove.bg API 处理


#### 显示处理进度提示（Loading 状态）


#### 处理完成后展示对比效果（原图 vs 处理后）


### 2.3 结果下载


#### 提供"下载图片"按钮


#### 下载格式：PNG（透明背景）


#### 文件命名：removed-bg-{timestamp}.png


### 2.4 错误处理


#### API 调用失败提示


#### 文件格式/大小不符合提示


#### API 额度用尽提示


## 三、技术实现


### 3.1 前端（Next.js App Router）


#### 页面：app/page.tsx


##### 上传区域（react-dropzone 或原生 input）


##### 图片预览区（原图 + 处理后对比）


##### 下载按钮


#### 样式：Tailwind CSS


#### 状态管理：React useState（无需复杂状态库）


#### 3.2 后端（Next.js API Routes）


##### 路由：app/api/remove-bg/route.ts


##### 功能：



###### 接收前端上传的图片（FormData）


###### 转发到 Remove.bg API


###### 返回处理后的图片（Base64 或 Blob）


##### 环境变量：REMOVE_BG_API_KEY



#### 3.3 部署（Cloudflare Pages）


##### 使用 Cloudflare Pages 部署


##### Edge Runtime 支持


##### 环境变量配置 API Key


## 四、非功能需求


### 4.1 性能


#### 图片上传响应时间 < 1s


#### 背景移除处理时间 < 5s（取决于 Remove.bg API）


#### 页面首屏加载 < 2s\


### 4.2 安全


#### 图片仅在内存中处理，不存储到服务器


#### API Key 存储在环境变量，不暴露到前端


#### 限制上传文件大小，防止滥用


### 4.3 用户体验


#### 响应式设计，支持移动端


#### 清晰的操作提示和错误信息


#### 简洁的 UI，无多余干扰
