# Image Background Remover

一个用于移除图片背景的工具。

## 功能

- 自动移除图片背景
- 支持多种图片格式（JPG、PNG、WEBP）
- 最大支持 10MB 文件

## 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Remove.bg API

## 使用方法

1. 安装依赖：`npm install`
2. 配置 `.env.local`，添加 Remove.bg API Key
3. 运行开发服务器：`npm run dev`
4. 访问 http://localhost:3000

## 部署

支持部署到 Cloudflare Pages
